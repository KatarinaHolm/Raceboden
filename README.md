# Raceboden

## About the project
An acquaintance of mine runs a company that rents out racing simulators to both events and private customers through time-based bookings. When I was studying Web Development 1 at Komvux, I saw it as a perfect opportunity to create a website for a real business.

I conducted an analysis of the target audience and considered what kind of feeling the website should convey. Since the business revolves around racing and speed, I chose colors and fonts associated with sports, cars, and energy. The design was intended to give a modern impression while remaining clear and easy to navigate.

An important part of the course focused on accessibility, which I have taken into great consideration in this project. The text is sufficiently large, the font is easy to read, and the color contrasts are adjusted to make the content clear for all users.
